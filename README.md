# 팀원 Tmi 카드 뒤집기 게임

## 🎤프로젝트 소개
각 팀원들의 사진으로 카드를 맞춰보는 게임입니다!<br>
모든 카드를 맞추면 팀원들의 TMI를 알 수 있을 거예요!<br>
* 장르 : 미니게임
* 아트 컨셉
